# AI-помощник по климатическим данным СП

> Telegram-бот + N8N + Kimi API + climate-sp-atlas. Инженер пишет название города — получает нормативные параметры по СП 20, СП 14, СП 131, СП 22 в структурированном виде.

---

## 🎯 Business Problem

При подготовке инженерных расчётов и коммерческих предложений менеджер или инженер должен быстро найти климатические параметры для объекта: снеговой район, ветер, гололед, сейсмичность, температуры, глубину промерзания.

**Раньше:** 10-15 минут ручного поиска по справочникам, риск ошибки при переносе данных в расчёты.

---

## ✅ What Was Done

**Архитектура:**
```
Telegram Bot → N8N Webhook → HTTP Request (GitHub JSON) → Code (поиск) → Kimi API → Telegram Reply
```

| Компонент | Роль |
|-----------|------|
| **Telegram Trigger** | Получает сообщение с названием города |
| **HTTP Request** | Скачивает settlements-climate.json (1096 записей) |
| **Code (JS)** | Ищет город в JSON по названию и региону |
| **Kimi API (Moonshot)** | Формирует человекочитаемый ответ по нормативам |
| **Telegram Reply** | Отправляет структурированный результат |

**Промпт для Kimi:**
```
Ты — инженер-консультант по строительным нормам. 
Сформируй краткий ответ с климатическими параметрами по СП. 
Используй только предоставленные данные.
```

**Инженер:** `Москва`

**Бот:**
```

📋 СП 20.13330
• Снеговой район: III (Sg = 1.8 кПа)
• Ветровой район: II (w0 = 0.30 кПа)
• Гололедный район: II

📋 СП 14.13330
• Сейсмичность: 6 баллов (ОСР-2015)

📋 СП 131.13330
• Среднегодовая температура: +5.8°C
• Холодный период: +3.5°C
• Теплый период: +18.1°C

📋 Глубина промерзания
• Глины: 1.35 м
• Пески: 1.60 м

---
🔗 Подробнее: https://kilevoy.github.io/climate-sp-atlas/
```

## 🛠 Stack

| Layer | Tech |
|-------|------|
| Workflow Engine | N8N (визуальный редактор, no-code) |
| LLM | Kimi 2.6 (Moonshot AI API, облачный) |
| Data Source | JSON-справочник climatesp-atlas (1096 населённых пунктов) |
| Messenger | Telegram Bot API |
| Деплой | N8N Cloud или Self-hosted |

---

## 📊 Business Impact

- **Время поиска:** с 10-15 минут до 3 секунд
- **Точность:** данные из проверенного JSON-справочника, ошибки исключены
- **Доступность:** Telegram доступен на любом устройстве
- **Масштабируемость:** 1096 городов в справочнике
