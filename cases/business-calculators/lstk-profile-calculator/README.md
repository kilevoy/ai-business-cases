# Автоматизация подбора и расчета ЛСТК-профилей

> Веб-инструмент для стандартизации расчета гнутых стальных профилей ПП, ПГС и ПZ: развертка, вес, цена, отход рулона и подбор более выгодных размеров через тепловую карту.

---

## Business Problem

В производстве гнутых ЛСТК-профилей много параметрических вариантов: тип профиля, толщина стали, высота стенки, ширина полок и отгибки. Полный прайс для всех комбинаций практически неудобен: менеджеру или проектировщику нужно быстро понять, сколько будет стоить конкретный профиль и насколько рационально он раскладывается по ширине рулона.

До автоматизации процесс зависел от ручных расчетов и согласований:

- параметры профиля передавались в производство или инженеру;
- отдельно считалась развертка заготовки;
- вручную проверялось, сколько заготовок помещается в рулон;
- отход и цена с учетом отхода пересчитывались в таблицах или вручную;
- подбор более выгодного размера требовал перебора вариантов.

Это увеличивало время подготовки ответа клиенту и создавало риск ошибки при переносе размеров, толщины, цены сырья или расчетного отхода.

---

## What Was Done

Создан веб-инструмент, который переводит расчет профиля в единый повторяемый сценарий:

| Feature | Description |
|---------|-------------|
| Типы профилей | Поддержаны ПП, ПГС и ПZ с разной геометрией изгибов |
| Толщины стали | 1.0 / 1.2 / 1.5 / 2.0 / 2.5 / 3.0 мм |
| Расчет развертки | Учитываются прямые участки, количество изгибов, внутренний радиус и нейтральный радиус |
| Расчет отхода | Считается количество заготовок из рулона, остаток в мм и процент отхода |
| Ценообразование | Показывается цена без отхода и цена с учетом отхода рулона |
| Подбор выгодного профиля | Тепловая карта показывает процент отхода по высоте стенки и полке A |
| Быстрая подстановка | Клик по ячейке карты подставляет выбранные размеры в основную форму |
| Отгибка C | Для ПГС и ПZ добавлен слайдер отгибки 13-27 мм, который пересчитывает карту |

---

## Process Automated

Автоматизирован участок от получения требуемой геометрии профиля до расчетной цены за погонный метр:

1. Менеджер или проектировщик выбирает тип профиля, толщину и размеры.
2. Система рассчитывает развертку заготовки с учетом геометрии гибки.
3. Система определяет, сколько заготовок помещается в рулон заданной ширины.
4. Рассчитывается отход рулона в миллиметрах и процентах.
5. Цена сырья за тонну переводится в цену профиля за погонный метр.
6. Тепловая карта помогает подобрать близкий размер с меньшим отходом.
7. Выбранный вариант переносится в расчет одним кликом.

## Architecture / Modules

| Module | Role |
|--------|------|
| `src/lib/calculator.ts` | Расчетная логика: геометрия профиля, развертка, отход, вес и цена |
| `src/lib/calculator.test.ts` | Unit-тесты для ключевых расчетов и проверки неподдерживаемых толщин |
| `src/App.tsx` | Интерфейс формы, расчет результата, тепловая карта подбора и синхронизация параметров |
| `src/index.css` | Базовые стили и Tailwind CSS |
| `.github/workflows/deploy.yml` | CI/CD: установка зависимостей, тесты, сборка и публикация на GitHub Pages |

Расчет вынесен из UI в отдельный модуль, поэтому бизнес-формулы можно тестировать отдельно от интерфейса.

## Business Impact

- **Снижение ручных ошибок**: формулы расчета развертки, веса, отхода и цены выполняются одинаково для каждого профиля.
- **Ускорение ответа клиенту**: менеджер видит цену и отход сразу после ввода параметров.
- **Стандартизация процесса**: единый сценарий расчета заменяет ручной перенос данных между производством, таблицами и коммерческим предложением.
- **Оптимизация отхода**: тепловая карта помогает подобрать размер профиля с меньшим процентом остатка рулона.
- **Прозрачность расчета**: на экране видны развертка, количество заготовок из рулона, отход, вес и две цены: без отхода и с отходом.
