# Калькулятор профилей ЛСТК (ПП / ПГС / ПZ)

> Интерактивный калькулятор раскроя гнутых стальных профилей ЛСТК с heatmap отходов. Шаг 1 мм → ∞ комбинаций → полный прайс невозможен.

---

## 🎯 Business Problem

Компания ИНСИ производит гнутые профили ЛСТК из оцинкованной рулонной стали. Станок позволяет изготавливать профили высотой **100–350 мм с шагом 1 мм** и полками **40–100 мм**.

**Проблема:** полный прайс невозможен — много комбинаций.

**Раньше:** для выяснения цены параметры отправлялись на производство. В компьютер станка задавались, производство сообщало менеджеру развёртку (заготовка какая нужна для производства профиля). Менеджер раскладывал ширину рулона на заготовки и вычислял отход.

---

## ✅ What Was Done

| Feature | Description |
|---------|-------------|
| **3 типа профилей** | ПП (П-образный, 2 изгиба), ПГС (с отгибками, 4 изгиба), ПZ (Z-образный, 4 изгиба) |
| **6 толщин** | 1.0 / 1.2 / 1.5 / 2.0 / 2.5 / 3.0 мм |
| **Инженерный расчёт** | НЕ простой подсчёт сумм — полная геометрия гибки с радиусами изгиба (СП 16.13330) |
| **Авторасчёт** | Развёртка, количество из рулона, отход мм / %, вес пог.м |
| **Ценообразование** | Цена без отхода и с учётом отхода в реальном времени |
| **Тепловая карта** | Карта отходов — проектировщик визуально выбирает оптимальный профиль |

**Для проектировщика — выбор профиля с минимальным отходом:**

Проектировщику важно не просто получить цену, а **выбрать профиль, который изготовят с минимальным отходом рулонной стали**. Калькулятор строит тепловую карту отхода для всех комбинаций высоты и ширины полки при заданной толщине:

- 🟢 Зелёный — отход ≤ 5% (идеально)
- 🟡 Жёлтый — отход 5-10% (хорошо)
- 🟠 Оранжевый — отход 10-20% (приемлемо)
- 🔴 Красный — отход > 20% (дорого)

Проектировщик кликает на ячейку карты — и размеры мгновенно подставляются в расчёт. За один взгляд видно, какие размеры выгодны, а какие удорожают производство.

**Инженерная глубина расчёта:**

```
Радиус внутреннего изгиба (rb):      по толщине из СП 16.13330
Нейтральный радиус (rcp):            rb + t / 2

Кол-во изгибов:
  • ПП = 2   • ПГС = 4   • ПZ = 4

Длина изогнутых участков (l0):
  l0 = bendsCount × (π × rcp / 2)

Длина прямолинейных участков (l1):
  • ПП:   H − 2×(rb + t) + 2×(A − (rb + t))
  • ПГС:  H − 2×(rb + t) + 2×(A − 2×(rb + t)) + 2×(C − (rb + t))
  • ПZ:   H − 2×(rb + t) + (A − 2×(rb + t)) + (B − 2×(rb + t)) + 2×(C − (rb + t))

Полная развёртка:                     l0 + l1
Количество из рулона:                 ⌊rollWidth / razvertka⌋
```

## 📊 Business Impact

- **Время подбора профиля** — с 15-20 минут до 1-2 минут
- **Точность расчёта** — исключены человеческие ошибки в сложной геометрии гибки
- **Прозрачность для клиента** — менеджер показывает расчёт на экране
- **Оптимизация отхода** — heatmap помогает снизить отход на 15-30%
