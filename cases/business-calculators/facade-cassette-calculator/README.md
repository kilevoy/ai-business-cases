# Калькулятор фасадных кассет и навесной фасадной системы

> Веб-инструмент для автоматизации расчета фасадных кассет, подсистемы, крепежа, утеплителя, комплектующих и итоговой стоимости проекта. Заменяет ручные Excel-расчеты и ускоряет подготовку коммерческого предложения.

---

## 🎯 Business Problem

Продажи фасадных систем требуют быстрого и точного расчета по заявке клиента: площадь фасада, раскладка кассет, тип крепления, проемы, углы, подсистема, утеплитель, мембраны, доборные элементы, саморезы и итоговая стоимость.

**Раньше:** менеджер или инженер собирал расчет вручную из Excel-файлов, прайса, технических альбомов и монтажных правил. При изменении размеров, типа кассет или прайса приходилось пересчитывать несколько связанных блоков.

**Проблема:** ручной процесс занимает время, зависит от опыта конкретного специалиста и создает риск ошибок при переносе данных между расчетом, прайсом и коммерческим предложением.

---

## ✅ What Was Done

Создано React/TypeScript-приложение, которое переводит расчет фасадных кассет в единый веб-сценарий: пользователь вводит параметры объекта, а система считает состав, количество и стоимость материалов.

| Feature | Description |
|---------|-------------|
| **Расчет фасадов** | Ввод размеров фасада, количества одинаковых фасадов и проемов с вычитанием из расчетной площади |
| **Типы кассет** | Поддержка КФ-1, КФ-2, КФ-3, КФ-4/30 и КФ-4/17 с разными правилами крепления и монтажной геометрии |
| **Раскладка кассет** | Автоматический расчет рядов, колонок, стандартных и доборных кассет по длине и высоте |
| **Подсистема** | Расчет элементов Г-образной, П-образной и двухуровневой подсистемы с учетом шага, углов и проемов |
| **Комплектующие** | Начальные планки, угловые кассеты, профили, кронштейны, крепеж, саморезы, заклепки |
| **Утеплитель и мембраны** | Опциональный расчет утеплителя, дюбелей и ветро-влагозащитной пленки |
| **Прайс из Excel** | Импорт актуального прайса, сопоставление позиций по кодам и fallback на встроенные цены |
| **Итоговая смета** | Сводная стоимость по группам материалов и расчет цены за квадратный метр |

## 🧩 Process Automated

Автоматизирован участок от первичной заявки клиента до расчетной базы для коммерческого предложения:

1. Ввод геометрии фасада и параметров облицовки.
2. Выбор типа кассеты, толщины, покрытия и монтажной схемы.
3. Расчет раскладки с учетом стандартных и доборных изделий.
4. Расчет подсистемы, утепления, крепежа и комплектующих.
5. Подтягивание цен из актуального прайса.
6. Формирование итоговой стоимости и цены за м².

## 🏗 Architecture

Расчетная логика вынесена в отдельные модели и справочники, чтобы интерфейс не был единственным местом хранения бизнес-правил.

| Module | Role |
|--------|------|
| `src/features/calculator/model/geometry.ts` | Геометрия фасада, проемов и расчетных областей |
| `src/features/calculator/model/price-upload.ts` | Импорт и нормализация Excel-прайса |
| `src/entities/catalog/*` | Справочники кассет, подсистемы, крепежа и ценовых позиций |
| `Документация/md/*` | Оцифрованные правила из СТО, альбомов технических решений и монтажных инструкций |
| `public/price.json` | Подготовленный прайс для веб-приложения |

## 📊 Business Impact

- **Скорость подготовки КП** — менеджер получает расчетную стоимость без ручного сбора данных из нескольких Excel-файлов.
- **Снижение ошибок** — единые правила расчета кассет, подсистемы, крепежа и комплектующих применяются автоматически.
- **Актуальность цен** — прайс можно обновлять через импорт Excel без переписывания расчетной логики.
- **Стандартизация процесса** — один инструмент для менеджера, инженера и производственного согласования.
- **Масштабируемость** — архитектура позволяет добавлять новые типы кассет, подсистем, прайсов и расчетных сценариев.
