# Калькулятор покрытия и переработки для ЛСТК

> Компаньон-калькулятор для расчёта итоговой цены тонны ЛСТК-профиля с порошковой окраской и переработкой.

---

## 🎯 Business Problem

После расчёта себестоимости сырья (в [lstk-profile-calculator](https://github.com/kilevoy/lstk-profile-calculator)) менеджеру нужно быстро добавить:
- Стоимость порошковой окраски (пропорционально площади)
- Стоимость переработки (гибка, резка)
- Получить итоговую цену за тонну

**Раньше:** ручной пересчёт в Excel, 10-15 минут, легко ошибиться в пропорциях.

---

## ✅ What Was Done

Простой но точный калькулятор:

```
Площадь тонны = (1000 / масса_пм) × (ширина_рулона / 1000) [м²]
Стоимость покрытия = площадь_тонны × цена_покрытия_за_м²
Итоговая цена = Цена_сырья + Стоимость_покрытия + Ставка_переработки
```

| Feature | Description |
|---------|-------------|
| **Вход** | Цена сырья, цена покрытия ₽/м², ставка переработки ₽/тонна |
| **Выход** | Итоговая цена/тонна по всему спектру толщин 0.5–3.0 мм |
| **Хранение** | LocalStorage — сохраняет настройки между сессиями |
