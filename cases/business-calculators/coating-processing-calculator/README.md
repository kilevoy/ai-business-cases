# Автоматизация расчета стоимости покрытия и переработки изделий

> Статический веб-инструмент для расчета итоговой цены за тонну продукции из оцинкованной рулонной стали с учетом сырья, порошковой окраски и ставки переработки.

---

## Business Problem

При продаже изделий из оцинкованной рулонной стали менеджеру нужно быстро рассчитать итоговую цену с учетом нескольких составляющих:

- цена сырья по каждой толщине;
- ширина рулона;
- масса погонного метра;
- стоимость порошковой окраски за квадратный метр;
- ставка переработки за тонну.

До автоматизации такой расчет приходилось делать вручную или в таблицах: отдельно считать площадь материала в одной тонне, переводить стоимость покрытия из рублей за м2 в рубли за тонну, затем добавлять сырье и переработку. При частом изменении цен сырья и ставок это создавало риск ошибок и замедляло подготовку коммерческих расчетов.

---

## What Was Done

Создан автономный веб-инструмент без серверной части. Пользователь вводит цены сырья, стоимость окраски и ставку переработки, а система формирует таблицу итоговых цен по всем доступным толщинам.

| Feature | Description |
|---------|-------------|
| Справочник материалов | 11 толщин от 0.5 до 3.0 мм с шириной рулона и массой погонного метра |
| Редактирование сырья | Цены сырья по толщине можно изменять в интерфейсе |
| Сохранение настроек | Цены сырья, ставка окраски, ставка переработки и состояние блока сохраняются в LocalStorage |
| Расчет покрытия | Стоимость окраски переводится из руб/м2 в руб/т через площадь материала в одной тонне |
| Расчет переработки | К итоговой цене добавляется ставка переработки за тонну |
| Таблица результата | Для каждой заполненной толщины показывается сырье, покрытие, цена без переработки и итоговая цена |
| Валидация | Отрицательные цены сырья блокируются перед расчетом |

---

## Process Automated

Автоматизирован процесс подготовки расчетной цены для продукции из рулонной оцинкованной стали:

1. Менеджер обновляет цены сырья по нужным толщинам.
2. Указывает стоимость порошковой окраски за квадратный метр.
3. Указывает ставку переработки за тонну.
4. Система рассчитывает, сколько квадратных метров приходится на одну тонну материала.
5. Стоимость окраски пересчитывается в рубли за тонну.
6. Система складывает сырье, покрытие и переработку.
7. Итоговая таблица используется как расчетная база для дальнейшего коммерческого предложения.

## Architecture / Modules

| Module | Role |
|--------|------|
| `index.html` | Структура страницы: блок цен сырья, параметры расчета и таблица результатов |
| `app.js` | Справочник материалов, состояние интерфейса, расчетные формулы, LocalStorage и рендер таблицы |
| `styles.css` | Визуальное оформление, адаптивная сетка, состояния редактирования и ошибок |

Ключевая формула в `app.js`:

```text
metersPerTonne = 1000 / mass
squareMetersPerTonne = metersPerTonne * (width / 1000)
coverageCostPerTonne = squareMetersPerTonne * paintingPrice
finalPrice = materialPrice + coverageCostPerTonne + processingRate
```

## Business Impact

- **Снижение ручных ошибок**: пересчет площади тонны и стоимости покрытия выполняется по одной формуле для всех толщин.
- **Ускорение подготовки расчетов**: менеджер получает таблицу цен по всему диапазону толщин после ввода актуальных ставок.
- **Стандартизация ценообразования**: сырье, покрытие и переработка учитываются в единой структуре.
- **Гибкость при изменении цен**: цены сырья можно обновить без изменения кода и повторного расчета в Excel.
- **Автономность**: инструмент работает как статическая страница и не требует серверной инфраструктуры.
