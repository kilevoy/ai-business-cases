# Калькулятор покрытия и переработки для ЛСТК

> Калькулятор итоговой цены тонны ЛСТК-профиля с учётом порошковой окраски и переработки. Работает в паре с [калькулятором профилей](https://kilevoy.github.io/gps/).

---

## 🎯 Business Problem

После расчёта себестоимости сырья менеджеру нужно быстро добавить:
- Стоимость порошковой окраски
- Стоимость гибки и резки
- Получить итоговую цену за тонну

**Раньше:** ручной пересчёт в Excel, легко ошибиться в пропорциях.

---

## ✅ What Was Done

Простой калькулятор:

```
Площадь тонны = (1000 / масса_пм) × (ширина_рулона / 1000) [м²]
Стоимость покрытия = площадь_тонны × цена_покрытия_за_м²
Итоговая цена = Цена_сырья + Стоимость_покрытия + Ставка_переработки
```

| Feature | Description |
|---------|-------------|
| **Вход** | Цена сырья, цена покрытия ₽/м², ставка переработки ₽/тонна |
| **Выход** | Итоговая цена/тонна по всему спектру толщин 0.5–3.0 мм |
| **Хранение** | LocalStorage — сохраняет настройки |
