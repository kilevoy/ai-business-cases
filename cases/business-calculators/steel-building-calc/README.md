# Калькулятор стального каркаса промышленных зданий

> Веб-приложение для подбора оптимальных стальных конструкций по СП 16.13330 / СП 20.13330. Заменяет 6 Excel-файлов и ускоряет подготовку КП на 30%.

---

## 🎯 Business Problem

Отдел продаж металлоконструкций готовит 50+ коммерческих предложений в неделю. Каждое КП требует:
- Ручного перебора 200+ профилей в 6 разных Excel-калькуляторах
- Проверки по СП 16.13330 (прочность, устойчивость, гибкость)
- Пересчёта нагрузок СП 20.13330 (ветер, снег, краны)
- Высокого риска человеческой ошибки → перерасход стали или отказ от КП

**Результат:** подготовка КП занимала 2-4 часа, частые ошибки, потеря заявок.

|--------|--------|-------|

## 🏗 Architecture

### 7 Вкладок (7 расчётных модулей)

| Вкладка | Что подбирает | Источник истины |
|---------|---------------|-----------------|
| Колонна | Стальной профиль колонны | Excel «Калькулятор колонн v6.1» |
| Ферма | Профили ВП, НП, ОРб, ОР, РР для фермы Молодечно | Excel «Фермы Молодечно v1.0» |
| Прогоны | Прокатные прогоны и ЛСТК (МП350, МП390, 2ТПС / 2ПС / Z) | Excel «Общий калькулятор прогонов v2.0» |
| Балка покрытия | ВБ, ГБ, колонны балочной клетки | Excel «Балочная клетка v3.1» |
| Оконные ригели | Нижний и верхний ригель | Excel «Оконные ригели v2.0» |
| Подкрановая балка | Профиль ПБ с проверками по СП 16.13330 | Excel «Подкрановая балка v2.0» |
| Сводка | Общая масса и стоимость каркаса | — |

### Ключевые решения

- **208 профилей × 4 марки стали (С255Б, С355Б, С245, С345) × 0–4 распорки = 2080 вариантов** на одну колонну
- **Полные проверки СП 16.13330:** σ, σ устойчивость X/Y, гибкость X/Y; для фермы — формулы 7, 44, 106, 109, 120
- **Нагрузки СП 20.13330:** снег, ветер с k(zₑ), ζ(zₑ), ν(ρ,χ) для местности A/B/C, кровля, ограждения
- **Общий BuildingContext:** пролёт, длина, высота, шаг рам, ветер, снег — синхронизированы между всеми вкладками
- **Автопередача нагрузок:** собственный вес прогонов и балок покрытия автоматически идёт в нагрузку на колонны
- **Подсчёт элементов здания:** рамы, крайние/средние/торцевые колонны, фахверк с учётом ГИП-правила для колонн с краном

## 📊 Verification Chain

Каждый расчётный модуль воспроизводит формулы Excel 1-в-1. Доказательная цепочка:

1. **`scripts/verify_baseline.py`** — Python-перевод формул колонной вкладки, проверен на default-сценарии (N=63.79 кН, M=143.90 кН·м)
2. **`scripts/excel_oracle.py`** — пересчёт исходного xlsx через LibreOffice для 10 сценариев, ground-truth в JSON
3. **`scripts/freeze_column_fixtures.py`** — фикстура `src/calc/__fixtures__/column-scenarios.json`
4. **`src/calc/*/engine.acceptance.test.ts`** — Vitest-тесты по фикстурам, регрессия ловится в CI

**Результат:** 0.00% расхождения по N и M между Python, Excel и TypeScript.
