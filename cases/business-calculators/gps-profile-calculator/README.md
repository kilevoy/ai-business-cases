# Калькулятор профилей ЛСТК (ПП / ПГС / ПZ) + Покрытие и переработка

> Два связанных калькулятора для расчёта себестоимости гнутых стальных профилей ЛСТК. Решают проблему бесконечного прайса: станок гнёт с шагом 1 мм, комбинаций — ∞.

---

## 🎯 Business Problem

Компания ИНСИ производит гнутые профили ЛСТК (лёгкие стальные тонкостенные конструкции) из оцинкованного рулона. Станок позволяет изготавливать профили с **шагом 1 мм** по ширине и высоте.

**Проблема:** комбинаторный взрыв → полный прайс невозможен:
- 26 вариантов высоты × 13 вариантов полки × 6 толщин × 3 типа профиля × отгибки = десятки тысяч комбинаций
- Менеджеру нужно назвать цену за время звонка
- Проектировщику нужно минимизировать отход рулонной стали
- Excel-таблицы не справляются — формулы сложные, ошибки дорогие

**Раньше:** 15-20 минут на один расчёт, ручной перебор, частые ошибки.

## ✅ What Was Done

### 🔥 Калькулятор 1: Раскрой и себестоимость профиля (`src/`)

| Feature | Description |
|---------|-------------|
| **3 типа профилей** | ПП (П-образный, 2 изгиба), ПГС (с отгибками, 4 изгиба), ПZ (Z-образный, 4 изгиба) |
| **6 толщин** | 1.0 / 1.2 / 1.5 / 2.0 / 2.5 / 3.0 мм |
| **Инженерный расчёт** | НЕ простой подсчёт сумм — полная геометрия гибки с радиусами изгиба (СП 16.13330) |
| **Авторасчёт** | Развёртка, количество из рулона, отход мм / %, вес пог.м |
| **Ценообразование** | Цена без отхода и с учётом отхода в реальном времени |
| **Тепловая карта** | Heatmap отхода — проектировщик визуально выбирает оптимальный профиль |

**Инженерная глубина расчёта:**

```
Радиус внутреннего изгиба (rb):      по толщине из СП 16.13330
Нейтральный радиус (rcp):            rb + t / 2

Кол-во изгибов:
  • ПП = 2   • ПГС = 4   • ПZ = 4

Длина изогнутых участков (l0):
  l0 = bendsCount × (π × rcp / 2)

Длина прямолинейных участков (l1):
  • ПП:   H − 2×(rb + t) + 2×(A − (rb + t))
  • ПГС:  H − 2×(rb + t) + 2×(A − 2×(rb + t)) + 2×(C − (rb + t))
  • ПZ:   H − 2×(rb + t) + (A − 2×(rb + t)) + (B − 2×(rb + t)) + 2×(C − (rb + t))

Полная развёртка:                     l0 + l1
Количество из рулона:                 ⌊rollWidth / razvertka⌋
```

**Параметры материала по толщине:**

| t (мм) | rb (мм) | Ширина рулона | Уд. вес (кг/м²) |
|---|---|---|---|
| 1.0 | 3.0 | 1250 | 7.5056 |
| 1.2 | 3.2 | 1250 | 9.2304 |
| 1.5 | 3.5 | 1250 | 11.852 |
| 2.0 | 4.0 | 1250 | 15.67696 |
| 2.5 | 4.5 | **1000** | 19.62 |
| 3.0 | 5.0 | **1000** | 23.5 |

### 🔥 Калькулятор 2: Покрытие и переработка (`next-calculator/`)

Простой калькулятор-компаньон для расчёта **итоговой цены тонны профиля** с учётом:
- Порошковой окраски (₽/м² покрытия)
- Переработки: гибка, резка (₽/тонна)

```
Площадь тонны = (1000 / масса_пм) × (ширина_рулона / 1000) [м²]
Стоимость покрытия = площадь_тонны × цена_покрытия_за_м²
Итоговая цена = Цена_сырья + Стоимость_покрытия + Ставка_переработки
```

| Параметр | Калькулятор 1 | Калькулятор 2 |
|----------|--------------|--------------|
| **Задача** | Какой размер и сколько стоит погонный метр | Сколько стоит тонна с покрытием |
| **Вход** | Размеры профиля + цена сырья | Цена сырья + цена покрытия + переработка |
| **Выход** | Развёртка, отход, цена/м | Итоговая цена/тонна по всем толщинам |
| **Стек** | TypeScript, React, Vite, Zod, vitest | Vanilla JS, CSS, LocalStorage |
| **Сложность** | 🟠 Средняя (геометрия гибки) | 🟢 Низкая (арифметика) |

## 📊 Business Impact

- **Время подбора профиля** — с 15-20 минут до 1-2 минут
- **Точность расчёта** — исключены человеческие ошибки в сложной геометрии гибки
- **Прозрачность для клиента** — менеджер показывает расчёт на экране
- **Оптимизация отхода** — heatmap помогает снизить отход на 15-30%
- **Цена с покрытием** — автоматический расчёт по всему спектру толщин
