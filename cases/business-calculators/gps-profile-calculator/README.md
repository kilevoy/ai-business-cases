# Калькулятор профилей ЛСТК (ПП / ПГС / ПZ)

> Веб-калькулятор для подбора оптимальных профилей лёгких стальных тонкостенных конструкций (ЛСТК) с расчётом отхода и ценообразованием в реальном времени.

---

## 🎯 Business Problem

Компания ИНСИ производит металлоконструкции из ЛСТК-профилей. Менеджерам по продажам нужно:
- Подбирать профили (ПП, ПГС, ПZ) под конкретный проект
- Рассчитывать отход металла при раскрое рулона
- Формировать цену с учётом реального отхода
- Делать это быстро при клиенте на встрече или в телефонном разговоре

**Раньше:** Excel-таблицы, ручной перебор, 15-20 минут на один подбор, частые ошибки в расчётах.

## ✅ What Was Done

| Feature | Description |
|---------|-------------|
| **3 типа профилей** | ПП (П-образный), ПГС (П-образный с отгибкой), ПZ (Z-образный) |
| **6 толщин** | 1.0 / 1.2 / 1.5 / 2.0 / 2.5 / 3.0 мм |
| **Авторасчёт** | Развёртка, количество из рулона, отход мм / %, вес пог.м |
| **Ценообразование** | Цена без отхода и с отходом в реальном времени |
| **Тепловая карта** | Визуальный подбор оптимального профиля по отходу (зелёный ≤5%, жёлтый 5-10%, оранжевый 10-20%, красный >20%) |
| **Интерактивность** | Клик по ячейке карты мгновенно подставляет размеры в форму |

## 🏗 Technical Details

### Типы профилей

| Тип | Полки | Отгибки | Формула развёртки |
|-----|-------|---------|-------------------|
| ПП | A = B | Нет | `H + 2×A` |
| ПГС | A = B | C (2 шт) | `H + 2×A + 2×C` |
| ПZ | A ≠ B | C (2 шт) | `H + A + B + 2×C` |

### Расчётные параметры
- **rb** — радиус загиба (зависит от толщины: 3.0–5.0 мм)
- **Ширина рулона** — 1250 мм (t ≤ 2.0) / 1000 мм (t ≥ 2.5)
- **Удельный вес** — 7.5–23.5 кг/м² в зависимости от толщины
- **Цена** — ручной ввод, руб/тонна

### Алгоритм
1. Выбор профиля и параметров
2. Расчёт развёртки с учётом радиуса загиба
3. Количество профилей из рулона = `floor(ширина рулона / развёртка)`
4. Отход = остаток рулона
5. Цена с отходом = `цена рулона / количество профилей`

## 📊 Business Impact

- **Время подбора профиля** — с 15-20 минут до 1-2 минут
- **Точность расчёта** — исключены человеческие ошибки в формулах
- **Прозрачность для клиента** — менеджер показывает расчёт на экране
- **Оптимизация отхода** — тепловая карта визуально показывает лучшие варианты
- **Портативность** — работает на телефоне, планшете, ноутбуке
