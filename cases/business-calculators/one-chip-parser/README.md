# Автоматизация конкурентной разведки и парсинга e-commerce каталога

> Многопоточный инструмент с GUI-интерфейсом для мониторинга цен, SKU-вариаций, характеристик и наличия товаров конкурента с целью динамического ценообразования и обогащения собственного каталога.

---

## Business Problem

В e-commerce ручной мониторинг конкурентов со сложной номенклатурой товаров (наличие множества SKU-вариаций внутри одной карточки: лицензии, аппаратные модули, кабели) приводит к критическим рискам:

- **Высокие трудозатраты**: ручной сбор цен и характеристик каталога на 1600+ позиций занимает десятки часов еженедельно;
- **Человеческий фактор**: регулярные ошибки при копировании сложных артикулов, цен для вариаций и статуса наличия;
- **Упущенная выгода**: задержки в изменении собственных цен ведут к снижению маржинальности или оттоку клиентов;
- **Замедление вывода новинок**: ручное перенесение подробных описаний и спецификаций сдерживает расширение собственного ассортимента на сайте.

---

## What Was Done

Разработано надежное настольное приложение, автоматизирующее весь цикл сбора и структурирования данных с целевого интернет-магазина автоэлектроники и оборудования для чип-тюнинга **one-chip.ru**.

| Feature | Description |
|---------|-------------|
| **BFS-обход дерева категорий** | Автоматический рекурсивный сбор структуры подкатегорий с фильтрацией дубликатов |
| **Многопоточный скрапинг** | Параллельный сбор данных о товарах в 8-16 потоков с помощью `ThreadPoolExecutor` |
| **Сложные SKU-вариации** | Извлечение списков опций внутри одной карточки (название опции, цена, артикул, наличие) |
| **Обогащение контента** | Извлечение ссылок на изображения, текстовых описаний и исходного HTML-кода разметки |
| **Отказоустойчивость** | Exponential Backoff при сетевых ошибках (до 4 попыток с растущей паузой) |
| **Умный догрузчик** | Фиксация сбойных URL в `failed.txt` и точечное докачивание в один клик с автослиянием данных |
| **Премиум-экспорт XLSX** | Генерация Excel-файла с листами по категориям, закреплением строк, фильтрами и автошириной колонок |

---

## Process Automated

Автоматизирован полный цикл конкурентной разведки и подготовки данных для импорта в CMS:

1. Пользователь выбирает корневые категории мониторинга и количество потоков в графическом окне.
2. Приложение собирает карту категорий и выстраивает очередь уникальных ссылок на товары.
3. Многопоточный движок выкачивает карточки товаров со скоростью **~10-15 карточек в секунду** (весь каталог за ~2 минуты).
4. Каждая карточка парсится на составляющие: базовый товар, его характеристики и вложенный список SKU-вариаций.
5. Данные конвертируются в плоский вид (одна вариация = одна строка) и сохраняются в форматы JSON, CSV и XLSX.
6. В случае сбоев сети создается лог ошибок и файл `failed.txt` для дозагрузки без повторного парсинга всего сайта.
7. Готовые файлы готовы к импорту в CRM/CMS или передаче в аналитические BI-системы.

## Architecture / Modules

| Module | Role |
|--------|------|
| `parser.py` | Ядро с логикой обхода BFS категорий, парсинга HTML, многопоточного скрапинга и сохранения в JSON/CSV/XLSX |
| `app.py` | GUI-интерфейс на Tkinter с живым логом, выбором параметров, прогресс-баром и интеграцией с failed.txt |
| `make_demo.py` | Скрипт генерации сокращенного демонстрационного каталога для демонстрации возможностей |
| `split_by_category.py` | Скрипт постобработки для разделения единого JSON по отдельным корневым CSV-таблицам |
| `build_exe.bat` | Автоматизация компиляции проекта в единый запускаемый `.exe` файл через PyInstaller |
| `run.bat` | Удобный скрипт для быстрого запуска GUI-приложения в ОС Windows |

## Business Impact

- **Ускорение процессов в 240 раз**: ручной мониторинг каталога (12 часов в неделю) заменен на 3-минутный автоматический прогон.
- **Динамическое ценообразование**: оперативный прайсинг на основе актуальных цен конкурента с детализацией до конкретной лицензии/модуля.
- **Управление ассортиментом**: отслеживание складских остатков конкурента для запуска таргетированной рекламы на временно дефицитные позиции.
- **Ускорение вывода новинок**: автоматическое наполнение карточек товаров на собственном сайте качественными описаниями и характеристиками.
